# 事件行为准则

问题：事件行为准则如何支持多元化和包容性？

## 描述
行为准则描述了事件参与者之间的良好行为规则，以及有人违反预期的良好行为时，可以采取哪些途径。 具有行为准则的事件表明组织者愿意对问题做出回应。

具有行为准则的事件表明组织者愿意对问题做出回应，这有助于各种背景的人感到包容并舒适地参与事件。 事件参与者有权报告问题，并主动帮助减轻存在不良行为的情况。

## 目标
- 事件组织者要确保提供有效流程来应对行为不端的参与者。
- 事件组织者要确保参与者在事件中获得积极体验。
- 事件参与者要知道如何举报攻击性行为。
- 事件参与者要知道他们在事件中是安全的。

## 实现

### 数据收集策略

- 采访和/或调查参与者，进一步了解事件的行为准则符合或不符合其预期的原因。
  * 事件如何改进行为准则？
  * 哪些示例可以说明事件达到或超过您对行为准则的预期？
  * 参与者在完成登记前是否需要接受行为准则？

- 查看事件网站的行为准则。

- 观察事件中是否发布了行为准则。

- 观察行为准则是否针对事件中的违规行为提供了明确的举报渠道。

- 观察行为准则/事件网站是否向不当行为受害者提供了可能的援助途径，并最终链接到外部机构？

- 浏览事件网站。 事件发布行为准则并且针对违规有明确的举报渠道，则符合标准。 （注意：理想情况下，行为准则应易于查找）

- 作为参与者或事件工作人员，要观察参与者是否容易找到事件中发布的行为准则。 建议在注册网站的突出位置发布行为准则。

- 行为准则相关的调查参与者：
  * 李克特量表 [1-x] 项：该事件在多大程度上满足了您对行为准则的预期。
  * 在注册和事件举办期间，您是否了解行为准则以及违规行为的举报途径？ [i]
  * 行为准则的存在是否让您觉得更安全、更有能力充分参与事件？ [i]
  * 如果您举报过违规行为，是否得到了满意的处理结果？ [i]
